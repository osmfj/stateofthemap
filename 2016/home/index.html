Conference

カンファレンス・テーマ

State of the Map Japan 2016は、自由な地図 “OpenStreetMap“ を作り、使い、育て、彩るひとびとの集いです。
今年のテーマは “リレーション“。
OpenStreetMapの名のもとに、集い、語らい、楽しみましょう！
Abstract

開催概要

  • 日程: 2016年 8月 6日（土） 10:00-17:00（9:30より受付開始）
  • 会場: 赤羽会館 大ホール・小ホール
  • 会場住所: 東京都 北区 赤羽南1-13-1 赤羽会館（OSMオブジェクトはこちら）
  • 最寄り駅: JR線・赤羽駅
  • 参加費: 無料（懇親会費別途） ※領収書がご入用の方は、当日会場受付にて発行をさせていただきます。
  • 定員: 200名（先着）
  • 主催: State of the Map 2016実行委員会
